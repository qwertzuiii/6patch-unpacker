<h2 align="center">How to use it?</h2>

`6patchunpacker.exe [ARGUMENTS]`

Arguments:
- `-f <FILEPATH (drag & drop file into cmd)>` - Select a file (NEEDED)
- `/create` or `/c` - Create a .6patch file
- `/nounpack` or `/nu` - No unpacking. Letting it as a zip
